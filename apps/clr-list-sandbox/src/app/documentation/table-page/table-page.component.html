Table properties

Property Type Description
dataProvider Table.Data.DataProvider Provides interface to load data to the table.
table TableDescription Define table UI representation: columns, sorting, etc in simple js object.
hidePaginator boolean (default false) Show / hide paginator for the table
disbaleSorting boolean (default false) Disable sorting for all columns even if they marked as sortable
aggregateRow Table.AggregateRow Display row with aggregated data from other rows. Data will be aggregated only for visible rows.
setFirstRowActiveOnFocus boolean (default true) Will set first row active when table component recieve focus.
rowSelectable boolean (default false) If this property set to true, then when some row clicked it will be highlighted as active. Usefull when open some item to display / edit in modal or sidepanel, this way it is clear for user which item from table is shown.
dragEnabled boolean (default false) Allow rows to be drag / dropped, see related drop event. TODO : Example
selectedRows (number | string)[] (default undefined) The list of the selected rows ids. If this value is undefined UI won't allow user select rows in the list, if this value is array (even empty), UI will display checkboxes which will allow user to select rows in the list. See related selectedRowsChanged event. See selected rows example.

Table events

Event Type Description
rowAction Table.RowActionEvent Will emit event when user click some item in row actions list. See related rowActions property on table defintion object .
cellClick Table.CellClickEvent Will emit event when user click on some row.
drop Table.DropEvent Will emit event when user dragged and dropped row. See related dragEnabled property.
