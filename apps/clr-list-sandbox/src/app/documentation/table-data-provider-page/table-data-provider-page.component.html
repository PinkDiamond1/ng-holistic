Table.Data.DataProvider

This interface is used by lists and tables in order to load data. The interface define single load method.

Load

    
        {{load_signature}}

State argument

Data structure which contains all required information to load list which may include sort order, pagination and filters. The shape of these data is not explicitly defined here, how they will be mapped to real request parameters is depends on application which implements this interface. See TableDataProviderConfig.mapState method for more explanation.

Return result

Observable stream which should return loaded data, the shape of these data is not explicitly defined here, how they will be mapped to list component models is defined in TableDataProviderConfig.mapResult method.
