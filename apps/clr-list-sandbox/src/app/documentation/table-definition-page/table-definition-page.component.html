TableDefinition

Property Type Description
cols (Table.Column | Table.CustomColumn | TMapColumns)[] List of the columns definitions see bellow. See example.
rowActions (optional) Table.RowAction[] | ((row: Table.Row) => Table.RowAction[]) Define list of actions associated with each row. Each row will have additional dropdown button component with provided actions. Instead of static array you could provide controller function which will take currently generated row and return array of the actions for each particular row, if this function will return empty array dropdown button won't be shown for the row. See example. This property intended to be used in conjuction with rowAction event
details (optional) TableDetails This property provides interafce to populate children rows of the parent row. See example.
sort (optional) string | Table.SortColumn Default sort column id (ascending order). With Table.SortColumn you could define not only column but also order. 
                    
                        {{sort_column}}

Table.Column

Property Type Description
id string Column identifier, when row provided to the table from json object, value from the object[id] property will be provided for this column.
title string Column title
sort (optional) string | boolean Define column as sortable. When column sortable it will have underline under the title to flag it as sortable. When user click this column, table will be reloaded with appropriate sort parameter. If this property is `true` then sort parameter will have value of column's `id` property. If this property is `string` then sort parameter will have exact this value.
cls (optional) string Class to apply on content of the column value.
whitespaceWrap (optional) boolean Allow wrap whitespace on cell content, off by default.
alignContent (optional) 'start' | 'center' | 'end' How to align text content inside the cell.
format (optional) (val: any, row: any) => string | ColumnFormat Returns mapped value given original value and the row. You could return `ColumnFormat` which contains not only mapped value itself but also class applied to the cell content. 
                    
                        {{column_format}}

TMapColumns

TMapColumns is the generic parameter which passed TableDefinition class and has default value of `Table.MapColumns.Column = LinkColumn | ImgColumn;` This column definition provides interface to insert into the cell arbitrary predifined components. By default there is defined 2 custom mapped columns `LinkColumn | ImgColumn`.

LinkColumn

Property Type Description
id, title, sort, cls - The same as in Table.Column
kind 'LinkColumn' The flag which signal library: for content of this column should be generated link component
props - Values to initilize properties of the link component
props.title string | (val: any, row: any) => string The text of the link. Also it could be a function which given col value and row returns appropriate title.
props.link (optional) string | (val: any, row: any) => string The url for the link. Also it could be a function which given col value and row returns appropriate title.
props.clicked (optional) Subject On each link click subject will recieve new event.

ImgColumn

Property Type Description
id, title, sort, cls - The same as in Table.Column
kind 'ImgColumn' The flag which signal library: for content of this column should be generated image component
props - Values to initilize properties of the image component
props.src string | (val: any, row: any) => string The path to the image. Also it could be a function which given col value and row returns appropriate src.
props.width (optional) string | (val: any, row: any) => string The width of the image (example '100px'). Also it could be a function which given col value and row returns appropriate width.
props.height (optional) string | (val: any, row: any) => string The heigth of the image (example '100px'). Also it could be a function which given col value and row returns appropriate height.

Table.RowAction

Property Type Description
id any Action id, this property could be used on event handling
title string Title of the action, it will be show on the dropdown item.

TableDetails

Property Type Description
cols Table.Column | Table.CustomColumn | Table.MapColumns.Column Subset of root cols (by id), if cols not in the subset they will be ignored.
rows (parentRow: Table.Row) => any[] Function which given parent row should return array of children rows. Then children rows will be generated as sub rows of the parent.
