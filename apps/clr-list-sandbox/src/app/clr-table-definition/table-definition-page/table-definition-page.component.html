TableDefinition

Property Type Description
cols (Table.Column | Table.CustomColumn | Table.MapColumns.Column)[] List of the columns definitions see bellow.
rowActions (optional) Table.RowAction[] | ((row: Table.Row) => Table.RowAction[]) List of actions associated with each row. UI will have additional first column with buttons, clicking by which will show dropdown with these actions.
details (optional) TableDetails This property provides interafce to populate children rows of the parent row.
sort (optional) string | Table.SortColumn Default sort column either id or reference to the column definition itself.

Table.Column

Property Type Description
id string Column identifier, when row provided to the table from json object, value from the object[id] property will be provided for this column.
title string Column title
sort (optional) string | boolean Define column as sortable. When column sortable it will have underline under the title to flag it as sortable. When user click this column, table will be reloaded with appropriate sort parameter. If this property is `true` then sort parameter will have value of column's `id` property. If this property is `string` then sort parameter will have exact this value.
cls (optional) string Class to apply on content of the column value.
whitespaceWrap (optional) boolean Allow wrap whitespace on cell content, off by default.
alignContent (optional) 'start' | 'center' | 'end' How to align text content inside the cell.
